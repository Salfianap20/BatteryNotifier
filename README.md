# Battery Alarm

Simple Windows battery monitoring application built with C# and WinForms.

## Features

* Battery percentage monitoring
* Customizable battery alarm threshold
* Sound notification when battery reaches target level
* System tray support
* Lightweight and easy to use

## Requirements

* Windows 10 / 11
* .NET Framework

---

## Installation

1. Download the latest release
2. Extract the ZIP file
3. Run `AdvancedBatteryNotifier.exe`

---

## Build From Source

1. Clone the repository

```bash
git clone https://github.com/Salfianap20/BatteryNotifier.git
```

2. Open the solution in Visual Studio
3. Build the project
4. Run the application

---

## Usage

1. Open the application
2. Set the desired battery percentage
3. Choose your preferred notification sound
4. Minimize to tray if needed
5. The alarm will play automatically when the battery reaches the configured percentage
6. If you want to try the application, you can download the file "BatteryAlarmSetup.exe".
